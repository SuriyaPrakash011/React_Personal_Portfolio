# Suriya Prakash M - Personal Portfolio

A stunning, responsive, and animated personal portfolio website built using **Next.js**, **Tailwind CSS**, and **Framer Motion**.

## 🚀 Live Demo
You can view the live site deployed on Vercel at:
**[https://suriyaprakash.vercel.app](https://suriyaprakash.vercel.app)**

## 🛠️ Features
- **Modern Tech Stack**: Built with Next.js, React, Tailwind CSS, and Framer Motion.
- **Dark/Light Mode**: Full theme customization with standard system preference detection and localStorage persistence.
- **Page Transitions**: Smooth, staggered colored curtains sliding across the screen on route changes.
- **Interactive Widgets**: Circular spinning text badge with hover effects.
- **Responsive Layout**: Designed to adapt perfectly to mobile, tablet, and desktop viewports.
- **Working Contact Form**: Connected to form submissions using FormSubmit to forward messages directly.

## 📁 Project Structure
- `src/pages/`: Next.js pages routing (Home, About, Projects, Articles).
- `src/components/`: Reusable interface elements (NavBar, Footer, Logo, TransitionEffect, HireMe, Layout, AnimatedText).
- `src/hooks/`: Custom state utility hooks (useThemeMode).
- `public/`: Project assets (profile image, project mockup images, resume PDF).
